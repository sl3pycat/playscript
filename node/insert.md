Node insert
---

### Cheatsheet
Method | Accepts | Description 
--- | --- | ---
insert | Any | Inserts children to the target node, an array inserts multiple children while a stringable value inserts it as text.
insert.html | String | Shorthand for `insert.htmlend`.
insert.htmlafter | String | Inserts HTML after the target.
insert.htmlbefore | String | Inserts HTML before the target.
insert.htmlend | String | Inserts HTML as the last child of the target.
insert.htmlstart | String | Inserts HTML as the first child of the target.
insert.text | String | Shorthand for `insert.textend`.
insert.textafter | String | Inserts text after the target.
insert.textbefore | String | Inserts text before the target.
insert.textend | String | Inserts text as the last child of the target.
insert.textstart | String | Inserts text as the first child of the target.
